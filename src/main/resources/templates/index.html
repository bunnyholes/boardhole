Spring Boot 게시판
학습 프로젝트

Spring Boot 3.5, Java 21을 기반으로 구현한 기본 게시판 시스템입니다. DDD 아키텍처, Spring Security 세션 인증, PostgreSQL/Redis를 활용한 학습용 프로젝트입니다. NoJS 기반 Thymeleaf 템플릿, CSS View Transitions, Popover API, @starting-style 등 최신 웹 기술을 지원하며, 라즈베리파이에서 운영됩니다.

아키텍처

계층형 아키텍처 구조

  • DDD 기반 계층형 아키텍처 (Presentation → Application → Domain → Infrastructure)
  • JPA/Hibernate, MapStruct, 커스텀 검증 어노테이션, Thymeleaf 템플릿
  • Docker Compose 기반 PostgreSQL/Redis 개발환경, 라즈베리파이 서버에서 최신 CSS API 지원

Spring Boot를 학습하기 위한 예제 프로젝트로, 실무에서 사용하는 기술 스택을 직접 구현해보며 배울 수 있습니다.

프로젝트 개요

스프링 부트 학습을 위한 구성

기본적인 CRUD 게시판 기능과 함께 사용자 인증, 권한 관리, 세션 처리 등 실무에서 필요한 기본 기능을 구현한 학습용 프로젝트입니다.

인증 시스템

Spring Security 폼 기반 로그인, Redis 세션 스토어, 역할 기반 권한 관리

게시판 기능

게시글 CRUD, 페이지네이션, 검색, 조회수 증가, 작성자 권한 체크

검증 & 예외처리

커스텀 검증 어노테이션, RFC 7807 Problem Details, 국제화 메시지

개발 환경

Docker Compose 자동 DB 구성, Testcontainers 통합 테스트, 라즈베리파이 서버 운영

ARCHITECTURE SNAPSHOT

boardhole 도메인 맵

  • Auth · User · Board · Shared

    각 도메인이 Presentation → Application → Domain → Infrastructure 레이어를 완비합니다.

  • 이벤트 처리

    게시글 조회 시 ViewedEvent를 통한 조회수 비동기 증가 처리

  • 품질 관리

    Spring Boot Actuator, 구조화된 로깅, API 문서화, 테스트 자동화
